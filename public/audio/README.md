# Áudio opcional

Coloque aqui um arquivo local chamado `ambient-tech.mp3` para ativar a trilha sonora da composição Remotion.

Sugestão criativa para a trilha:

- hum eletrônico suave;
- glitches digitais discretos;
- whooshes leves sincronizados com a passagem das palavras;
- atmosfera tecnológica premium e corporativa.

O arquivo `ambient-tech.mp3` não deve ser commitado. A raiz do projeto já contém `.gitignore` bloqueando MP3, WAV, FLAC, AAC e OGG dentro desta pasta para evitar binários pesados no repositório.
