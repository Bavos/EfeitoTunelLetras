# Audio design

Esta pasta documenta a direção sonora da peça, sem versionar binários.

## Ambiência recomendada

- Hum eletrônico grave e suave, entre -28 dB e -22 dB.
- Glitches digitais discretos em pontos de passagem das palavras.
- Whooshes leves sincronizados com o dolly-in da câmera.
- Textura corporativa premium, sem agressividade gamer/cyberpunk.

## Entrega para render

Exporte um MP3 estéreo de 8 segundos em `public/audio/futuristic-ambience.mp3` e renderize usando `--props` conforme o README principal.
